# Sanlyn OS

Smart Logistics Platform for FortuneSanlyn

## Features
- Multi-role login (Admin / Customer / Factory / Driver)
- Order management with real-time status
- Document confirmation workflow (B/L, Certificate, PI, Payment)
- AI-powered payment receipt OCR
- Task board with priority management
- Product news & promotions

## Tech Stack
- Next.js 14
- React 18
- Deployed on Vercel

## Getting Started
```bash
npm install
npm run dev
```
